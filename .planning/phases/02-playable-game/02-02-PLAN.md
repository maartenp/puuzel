**src/game/state.rs** — Add mutation methods to `PuzzleState`:

`pub fn handle_cell_click(&mut self, row: usize, col: usize)`:
- If clicked cell is Black, ignore
- If clicked cell == current `selected_cell`: toggle `selected_direction` (Across<->Down) per D-10/INTR-02
- Else: set `selected_cell = Some((row, col))`, set `selected_direction` to the direction of the word containing this cell (prefer Across if both; per INTR-06). If cell is at the start of both an across and down word, prefer the one that was NOT previously selected direction.

`pub fn set_letter_and_advance(&mut self, ch: char)`:
- If no `selected_cell`, return
- Set `user_grid[row][col] = Some(LetterToken::Single(ch.to_ascii_uppercase()))` per INTR-03
- Advance cursor: move to next white cell in `selected_direction`. Skip Black cells. If at end of word, stop (don't wrap).

`pub fn handle_ij_input(&mut self)`:
- Special case for IJ digraph: if the current cell's answer is `LetterToken::IJ` and `user_grid[row][col] == Some(LetterToken::Single('I'))`, promote to `Some(LetterToken::IJ)` and advance.
- This is called when 'J' is typed after 'I' in a cell that expects IJ.

`pub fn backspace(&mut self)`:
- If no `selected_cell`, return
- If `user_grid[row][col]` is Some: clear it to None per D-11/INTR-04
- If already None: move back one cell in `selected_direction` (skip Black), clear that cell

`pub fn move_cursor(&mut self, direction: Direction, delta: i32)`:
- Move selection by delta cells in the given direction per D-12
- Skip Black cells. Clamp to grid bounds.
- Update `selected_direction` to match movement direction

`pub fn cycle_clue(&mut self, delta: i32)`:
- Cycle through clues in order per D-12 (Tab/Shift+Tab)
- Build combined list: all across clues sorted by number, then all down clues sorted by number
- Find current active clue index, add delta (wrap around)
- Select the first cell of the new clue's slot, set direction accordingly

`pub fn select_clue(&mut self, slot: &Slot)`:
- Set `selected_direction = slot.direction`
- Find first empty cell (user_grid is None) in the slot; if all filled, select the first cell
- Set `selected_cell` to that cell per D-08/INTR-05

**src/input/handler.rs:** Implement `pub fn process_input(state: &mut PuzzleState, layout: &GridLayout)`:

```rust
use macroquad::prelude::*;

pub fn process_input(state: &mut PuzzleState, layout: &GridLayout) {
    // Keyboard: letter input per D-10/INTR-03
    while let Some(ch) = get_char_pressed() {
        if ch.is_alphabetic() {
            let upper = ch.to_ascii_uppercase();
            // IJ handling: if 'J' typed and current cell has 'I' and answer is IJ
            if upper == 'J' {
                // Check if current cell user entry is Single('I') and answer is IJ
                state.handle_ij_input();
                // If handle_ij_input consumed it, continue; otherwise treat as normal J
                // (Implementation detail: handle_ij_input returns bool to indicate consumption)
            }
            state.set_letter_and_advance(upper);
        }
    }

    // Keyboard: backspace per D-11/INTR-04
    if is_key_pressed(KeyCode::Backspace) {
        state.backspace();
    }

    // Keyboard: arrow keys per D-12
    if is_key_pressed(KeyCode::Right) { state.move_cursor(Direction::Across, 1); }
    if is_key_pressed(KeyCode::Left)  { state.move_cursor(Direction::Across, -1); }
    if is_key_pressed(KeyCode::Down)  { state.move_cursor(Direction::Down, 1); }
    if is_key_pressed(KeyCode::Up)    { state.move_cursor(Direction::Down, -1); }

    // Keyboard: Tab/Shift+Tab per D-12
    if is_key_pressed(KeyCode::Tab) {
        if is_key_down(KeyCode::LeftShift) || is_key_down(KeyCode::RightShift) {
            state.cycle_clue(-1);
        } else {
            state.cycle_clue(1);
        }
    }

    // Mouse: cell click per D-10/INTR-01
    if is_mouse_button_pressed(MouseButton::Left) {
        let (mx, my) = mouse_position();
        if let Some((row, col)) = layout.hit_test(mx, my, state.grid.height, state.grid.width) {
            state.handle_cell_click(row, col);
        }
    }
}
```
